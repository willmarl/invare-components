Low power battery-backed Real-Time Clock (RTC). Uses I2C communication to provide seconds, minutes, hours, day, date, month, and year information valid up to 2100. Uses coin battery to power keeping track while rest of device is turned off. Typically lasts ~5 years. Contains 56 bytes general purpose memory SRAM (can store any data you want).

- Operating Voltage : 4.5V - 5.5V
